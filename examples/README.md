# Hermetic Examples

Working examples for the three integration paths.

## Quick Start (Recommended)

```bash
hermetic init --quickstart
```

This handles everything: vault creation, .env import, daemon startup, and MCP configuration.

## Examples

| Example | Path | Description |
|---------|------|-------------|
| MCP Integration | [mcp-quickstart.md](mcp-quickstart.md) | Connect Claude Code, Cursor, or Windsurf |
| Python SDK | [sdk-quickstart.py](sdk-quickstart.py) | Make authenticated requests from Python |
| CLI Workflow | [cli-workflow.sh](cli-workflow.sh) | Full lifecycle via command line |

## Prerequisites

- Hermetic installed (`curl -sSf https://hermeticsys.com/install.sh | sh`)
- At least one API key to store
- Linux (Ubuntu 24.04 tested)
